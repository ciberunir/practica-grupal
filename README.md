# Documentación de Ataque de Phishing con WiFi Pineapple

Página web de documentación sobre ataques de phishing utilizando WiFi Pineapple con portales cautivos.

## Descripción

Este proyecto documenta un ataque de phishing mediante el uso de un dispositivo WiFi Pineapple con un portal cautivo que simula el login de un servicio web. El contenido es exclusivamente educativo y tiene como objetivo concienciar sobre los riesgos de seguridad en redes WiFi públicas.

## Contenido del Sitio

El sitio web incluye las siguientes secciones:

1. **Conceptos Básicos**: Explicación de los fundamentos del ataque, incluyendo qué es el phishing, WiFi Pineapple, portales cautivos y ataques Evil Twin.

2. **Prueba de Concepto**: Documentación paso a paso del proceso de configuración y ejecución de un ataque de phishing con portal cautivo.

3. **Variaciones**: Diferentes variantes del ataque, como ataques con deautenticación, Karma Attack, SSL Strip, entre otros.

4. **Protección**: Guía completa de medidas de protección y buenas prácticas para protegerse frente a este tipo de ataques.

5. **Bibliografía**: Referencias y recursos de interés sobre seguridad WiFi y técnicas de phishing.

6. **Autores**: Información sobre el equipo que desarrolló el proyecto.
